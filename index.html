Our Difficulties:

In this project we had to write a lot of JavaScript code for the first time, and we found it very hard to

write the classes and functions in a readable way, the code was in a lot of mess most of the time.

Another difficulty was to implement the validation plugin in the registration window, this took us a lot of time

to do. Even though we had a lot of fun writing this game, this was one of the most enjoying projects

we worked on.


We have used jQuery version 3.2.0, and in the validation plugin which helped us during the

validation in the registration page. In addition, we used a template from W3 school

in order to create the left navigation menu.
